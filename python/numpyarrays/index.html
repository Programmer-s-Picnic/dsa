2. Indexing: finding one item

Indexing means selecting one item from a sequence. In Python, indexing starts from 0, not from 1.

index 0
10
index 1
20
index 2
30
index 3
40
index 4
50
numbers = [10, 20, 30, 40, 50]

print(numbers[0])   # 10
print(numbers[1])   # 20
print(numbers[4])   # 50

Negative indexing

Negative indexing starts from the end. The last item is -1. 

numbers = [10, 20, 30, 40, 50]

print(numbers[-1])  # 50
print(numbers[-2])  # 40
print(numbers[-5])  # 10
Common mistake: if a list has 5 items, the last positive index is 4. numbers[5] will give an error.
Checkpoint

For marks = [75, 80, 92, 67], what is marks[2]?

Answer: 92.

3. Slicing: taking a part of a sequence

Slicing means selecting a range of values.

list[start : stop : step]

Important rule: start is included, but stop is not included. 

numbers = [10, 20, 30, 40, 50, 60]

print(numbers[1:4])    # [20, 30, 40]
print(numbers[:3])     # [10, 20, 30]
print(numbers[3:])     # [40, 50, 60]
print(numbers[::2])    # [10, 30, 50]
print(numbers[::-1])   # [60, 50, 40, 30, 20, 10]

Slicing strings

Strings also support indexing and slicing. 

name = "Champak"

print(name[0])      # C
print(name[1:4])    # ham
print(name[::-1])   # kapmahC
Beautiful idea: indexing gives one value, slicing gives a collection of values.

4. Dictionaries: storing data with names

A dictionary stores data as key-value pairs. Instead of remembering position numbers, we can use meaningful names. 

student = {
    "name": "Aarav",
    "math": 86,
    "science": 91,
    "english": 78
}

print(student["name"])
print(student["math"])

Why dictionaries matter in data science

Real data usually has labels:

Key Value Meaning
"name" "Aarav" Name of the student
"math" 86 Marks in mathematics
"science" 91 Marks in science

Dictionary operations

student = {
    "name": "Aarav",
    "math": 86,
    "science": 91
}

# Reading
print(student["math"])

# Adding
student["english"] = 78

# Updating
student["math"] = 90

# Safe reading
print(student.get("history", "Not found"))

# Looping
for key, value in student.items():
    print(key, "=", value)
Lists are good when order is important. Dictionaries are good when names are important.

5. NumPy Arrays: lists become powerful

NumPy means Numerical Python. It gives us arrays that are faster and more powerful than normal Python lists for numerical work.

Normal list

marks = [70, 80, 90, 60]

print(marks)

NumPy array

import numpy as np

marks = np.array([70, 80, 90, 60])

print(marks)

Why NumPy arrays are useful

With normal lists, mathematical operations are not direct. 

numbers = [1, 2, 3]

print(numbers * 2)
# Output: [1, 2, 3, 1, 2, 3]

With NumPy arrays, mathematical operations happen value by value. 

import numpy as np

numbers = np.array([1, 2, 3])

print(numbers * 2)
# Output: [2 4 6]
Important: NumPy thinks numerically. Lists think generally.

6. Shape: understanding the size of data

Shape tells us the structure of an array.

One-dimensional array

import numpy as np

a = np.array([10, 20, 30, 40])

print(a.shape)

Output: 

(4,)

This means: one row-like line with 4 values.

Two-dimensional array

import numpy as np

marks = np.array([
    [80, 85, 90],
    [70, 75, 65],
    [92, 88, 95]
])

print(marks.shape)

Output: 

(3, 3)

This means: 3 rows and 3 columns.

Shape Meaning
(4,) One-dimensional array with 4 values
(3, 2) Two-dimensional array with 3 rows and 2 columns
(2, 3, 4) Three-dimensional array
Shape is one of the most important ideas in Machine Learning. If the shape is wrong, the model may not train.

7. Axis: direction of calculation

Axis tells NumPy which direction to calculate in. 

import numpy as np

marks = np.array([
    [80, 85, 90],
    [70, 75, 65],
    [92, 88, 95]
])

print(marks)

Think of this as:

Student Math Science English
Student 1 80 85 90
Student 2 70 75 65
Student 3 92 88 95

axis = 0

axis=0 means calculate column-wise. It moves down the rows. 

print(np.mean(marks, axis=0))

This gives the mean of each subject.

axis = 1

axis=1 means calculate row-wise. It moves across the columns. 

print(np.mean(marks, axis=1))

This gives the mean of each student.

Memory trick: axis=0 collapses rows and gives column results. axis=1 collapses columns and gives row results.

8. Practice: mean, median, min, max, range

Now we start thinking numerically.

Dataset

import numpy as np

data = np.array([12, 18, 20, 25, 30, 35, 40])

Mean

Mean is the average value. 

mean_value = np.mean(data)
print("Mean:", mean_value)

Median

Median is the middle value after sorting. 

median_value = np.median(data)
print("Median:", median_value)

Minimum and Maximum

min_value = np.min(data)
max_value = np.max(data)

print("Minimum:", min_value)
print("Maximum:", max_value)

Range

Range means maximum minus minimum. 

range_value = np.max(data) - np.min(data)
print("Range:", range_value)

Full program

import numpy as np

data = np.array([12, 18, 20, 25, 30, 35, 40])

mean_value = np.mean(data)
median_value = np.median(data)
min_value = np.min(data)
max_value = np.max(data)
range_value = max_value - min_value

print("Data:", data)
print("Mean:", mean_value)
print("Median:", median_value)
print("Minimum:", min_value)
print("Maximum:", max_value)
print("Range:", range_value)
This is the beginning of data analysis. We are not only printing numbers. We are asking questions about the numbers.

9. Two-dimensional statistics

Now let us use a 2D array. Imagine 4 students and 3 subjects. 

import numpy as np

marks = np.array([
    [80, 85, 90],
    [70, 75, 65],
    [92, 88, 95],
    [60, 72, 68]
])

print("Marks:")
print(marks)

print("Shape:", marks.shape)

print("Overall mean:", np.mean(marks))
print("Subject-wise mean:", np.mean(marks, axis=0))
print("Student-wise mean:", np.mean(marks, axis=1))

print("Subject-wise minimum:", np.min(marks, axis=0))
print("Subject-wise maximum:", np.max(marks, axis=0))
print("Subject-wise range:", np.max(marks, axis=0) - np.min(marks, axis=0))

Meaning of results

Code Meaning
np.mean(marks) Mean of all numbers
np.mean(marks, axis=0) Mean of each subject
np.mean(marks, axis=1) Mean of each student
np.max(marks, axis=0) - np.min(marks, axis=0) Range of each subject
