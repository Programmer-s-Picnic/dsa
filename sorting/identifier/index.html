How to enter array changes

Use one array per line. Labels can be Start, Pass 1, Step 1, Round 1, etc.

Start: [5,1,4,2,8]
Pass 1: [1,4,2,5,8]
Pass 2: [1,2,4,5,8]
Bubble Sort family

Bubble Sort repeatedly swaps adjacent elements. In standard Bubble Sort, the largest unsorted value moves to the right after each pass.

Standard Bubble Optimized Bubble Cocktail Shaker

Selection Sort family

Selection Sort selects the smallest remaining value and places it at the front. Double Selection places both minimum and maximum in one round.

Standard Selection Double Selection

Insertion Sort family

Insertion Sort grows a sorted left side. Binary Insertion uses binary search to locate the insertion position. Gnome Sort behaves like an insertion-style swap walk.

Standard Insertion Binary Insertion Gnome Sort

Important limitation

Small arrays may produce the same pass for more than one algorithm. In that case, use the confidence scores and provide more passes for a better match.
