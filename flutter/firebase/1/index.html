🔥 Flutter + Firebase Lesson

Hello World Android Flutter App Connected to Firebase

In this lesson, we will create a simple Android Flutter app, connect it with Firebase, initialize Firebase in code, and display a success message on the screen.

Level: Beginner Platform: Android Tool: Flutter Backend: Firebase

What you will build

  • A Flutter Android project
  • A Firebase project
  • Firebase connection setup
  • A working Hello World screen
  • A simple Firebase connection test

Flutter

Flutter is used to build the mobile app interface.

Firebase

Firebase gives backend services like authentication, database, storage, and notifications.

Android

We will run this app on an Android emulator or real Android phone.

1

Create a Flutter Project

Open terminal and run:

flutter create hello_firebase_app
cd hello_firebase_app
2

Create Firebase Project

Go to Firebase Console and create a new project.

Simple idea: Firebase project is like the backend home of your app.
3

Add Android App in Firebase

Inside Firebase, add an Android app. Use your Flutter package name. 

com.example.hello_firebase_app

Then download:

google-services.json

Place it here:

android/app/google-services.json
4

Add Firebase Package

Open pubspec.yaml and add:

dependencies:
  flutter:
    sdk: flutter

  firebase_core: ^latest_version

Then run:

flutter pub get
5

Recommended Modern Setup

The modern FlutterFire method is better because it creates firebase_options.dart automatically. 

dart pub global activate flutterfire_cli
flutterfire configure
6

Complete main.dart Code

Replace lib/main.dart with this:

import 'package:flutter/material.dart';
import 'package:firebase_core/firebase_core.dart';
import 'firebase_options.dart';

void main() async {
  WidgetsFlutterBinding.ensureInitialized();

  await Firebase.initializeApp(
    options: DefaultFirebaseOptions.currentPlatform,
  );

  runApp(const MyApp());
}

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Hello Firebase App',
      debugShowCheckedModeBanner: false,
      theme: ThemeData(
        colorSchemeSeed: Colors.orange,
        useMaterial3: true,
      ),
      home: const HomePage(),
    );
  }
}

class HomePage extends StatelessWidget {
  const HomePage({super.key});

  @override
  Widget build(BuildContext context) {
    final firebaseAppsCount = Firebase.apps.length;

    return Scaffold(
      appBar: AppBar(
        title: const Text('Programmer\\'s Picnic'),
        centerTitle: true,
      ),
      body: Center(
        child: Card(
          margin: const EdgeInsets.all(24),
          elevation: 4,
          child: Padding(
            padding: const EdgeInsets.all(28),
            child: Column(
              mainAxisSize: MainAxisSize.min,
              children: [
                const Icon(
                  Icons.local_fire_department,
                  size: 72,
                  color: Colors.orange,
                ),
                const SizedBox(height: 16),
                const Text(
                  'Hello World + Firebase',
                  style: TextStyle(
                    fontSize: 26,
                    fontWeight: FontWeight.bold,
                  ),
                  textAlign: TextAlign.center,
                ),
                const SizedBox(height: 12),
                Text(
                  'Firebase apps connected: $firebaseAppsCount',
                  style: const TextStyle(fontSize: 18),
                ),
                const SizedBox(height: 12),
                const Text(
                  'If this screen opens without error, Firebase is initialized successfully.',
                  textAlign: TextAlign.center,
                ),
              ],
            ),
          ),
        ),
      ),
    );
  }
}
7

Run the App

Connect your Android phone or open an emulator, then run:

flutter run
Expected output: Hello World + Firebase

Very Simple Explanation

main()

The app starts from here.

WidgetsFlutterBinding

Prepares Flutter before Firebase starts.

Firebase.initializeApp()

Connects the app to Firebase.

MaterialApp

Creates the base Flutter app.

Scaffold

Creates the screen structure.

Firebase.apps.length

Checks how many Firebase apps are initialized.

Setup Checklist

Common Errors and Fixes

Error: Firebase has not been correctly initialized

Fix: Make sure Firebase.initializeApp() is called before runApp().


Error: firebase_options.dart not found

Fix: Run flutterfire configure.


App builds but Firebase does not work

Fix: Check package name and Firebase Android app configuration.
